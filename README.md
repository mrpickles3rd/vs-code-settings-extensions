# vs-code-settings-extensions
A backup of my setting and extensions for VS-Code

#### Exstentions:
* Babel ES6/ES7 -> May not be needed.
* Beautify -> May not be needed.
* ESLint
* Git History (git log)
* Jest
* Jest Snippets
* Markdown All in One
* Markdown Preview Enhanced
* npm Intellisense
* TODO Highlight
